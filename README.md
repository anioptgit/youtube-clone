# 🎬 YouTube Clone

A simple YouTube homepage clone built using HTML and CSS.  
Focused on layout design, styling, and creating a clean user interface.

## 🎗️ Features
- Header with search bar
- Sidebar navigation
- Video grid layout
- Clickable thumbnails and titles
- Clean UI with basic styling
- Use of icons for better visual appearance

## 🛠️ Tech Stack
- HTML
- CSS
